Cloner un repository :

git clone <url_du_repository> : Cette commande permet de cloner un repository existant dans un nouveau dossier.

Les 3 commandes de push :

1)   Ajouter les fichiers modifiés au staging area en utilisant la commande git add :
git add <nom_du_fichier>

Cette commande permet d'ajouter les fichiers modifiés ou nouvellement créés à la staging area.

2)  Effectuer un commit en utilisant la commande git commit :
git commit -m "Message de commit"

Cette commande permet d'enregistrer les modifications effectuées dans le repository local avec un message de commit spécifié.

3) Pousser les modifications vers le repository distant en utilisant la commande git push :
git push <nom_du_remote> <nom_de_la_branche>

Cette commande permet de pousser les modifications enregistrées localement vers le repository distant spécifié.
